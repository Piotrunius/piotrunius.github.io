## ğŸŒ Deployment

The site is automatically deployed via GitHub Pages from the `main` branch. Any commits to the main branch trigger an automatic deployment.

### Local Development

1. Clone the repository: 
```bash
git clone https://github.com/Piotrunius/piotrunius.github.io.git
```

2. Open `index.html` in your browser or use a local server: 
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve . 
```

3. Navigate to `http://localhost:8000`
